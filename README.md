# install_sysreptor
**install_sysreptor** is a bash script that automates [SysReptor](https://github.com/Syslifters/sysreptor) installation on Debian-based distributions like Kali, Ubuntu, Parrot as well as Debian itself. It also allows you to download HTB and Offsec exams' template and import them in SysReptor so that you can use them directly.  

# Installation
The script can easily be installed with the following commands:  
```
git clone https://github.com/0liverFlow/install_sysreptor
```
```
cd install_sysreptor
```
```
chmod +x install_sysreptor.sh
```
```
sudo ./install_sysreptor.sh
```
